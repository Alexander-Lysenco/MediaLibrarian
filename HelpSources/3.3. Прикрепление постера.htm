Прикрепление постера к записи

Постер - это художественно оформленный плакат, используемый для рекламных или декоративных целей. В большинстве своем постер является обложкой некоего рекламируемого продукта. В программе "Медиа-библиотекарь" есть возможность создать запись вместе с этой обложкой.

Для прикрепления постера во время добавления или редактирования записи, воспользуйтесь полем "Постер":

Постер можно выбрать из файловой системы Вашего компьютера. Рекомендуемые форматы - JPEG(JPG), PNG. Допускается прикрепление постера в форматах BMP и GIF. При прикреплении изображения с прозрачным фоном возможны проблемы с его дальнейшим отображением, поскольку изображения при сохранении конвертируются в JPG для удобства индексации и экономии места. В случае успешного прикрепления появится синяя надпись "Выбран локальный файл", рядом появится маленькое превью приклепленной картинки:

Постер можно загрузить в программу по прямой ссылке из Интернета. Поддерживаемые протоколы - HTTP, HTTPS, FTP. Допускаются форматы изображений JPG, PNG, BMP, GIF. Рекомендуется копировать прямую ссылку и вставлять ее в поле. Не рекомендуется набирать ссылку руками, поскольку при изменении текста в поле прикрепления, программа постоянно проверяет доступность введенного адреса, при медленном соединении с Интернет программа просто повиснет. В случае, если введенный адрес недоступен, появится синяя надпись с сообщением об ошибке:

В случае успешной загрузки изображение надпись сменится на "Изображение загружено", и появится превью изображения.

После нажатия кнопки "Сохранить" постер прикреплен к записи. можно посмотреть его предпросмотр из главного окна программы или развернуть его на весь экран при нажатии на превью в главном окне:

Если запись редактируется, и постер ей больше не требуется, для его удаления потребуется всего лишь очистить текст поля прикрепления постера, и/или оставить его пустым. В этом случае, после сохранения записи постер удалится.
Стоит отметить, что все сохраненные постеры хранятся в папке с программой, в подпапке с названием библиотеки и именем файла, совпадающим с именем записи, к которой приклеплен. При изменении названия записи, изменяется и имя этого файла. Во время редактирования записи поле прикрепления постера не хранит оригинальный источник постера - там будет указан путь к постеру, сохраненному в папке с программой. При удалении записи удаляется этот файл из подпапки для библиотеки. При удалении/очистке библиотеки удаляется вся подпапка с постерами. Напомним, что изображения при сохранении конвертируются в формат JPG для удобства индексации и экономии места, поэтому возможны проблемы с отображением изображений с прозрачным фоном.
