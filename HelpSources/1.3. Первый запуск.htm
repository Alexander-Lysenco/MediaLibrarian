При запуске Вас встретит окошко загрузки программы с красивым логотипом:

Через некоторое время появится и основное окно программы:

Рассмотрим разделы интерфейса этого окна:

Строка меню с базовым функционалом программы. В меню Файл можно выбрать библиотеку, очистить выбранную библиотеку или закрыть программу. Меню Правка связано с кнопками действия над библиотекой: добавления записей, редактирования, удаления и поиска. В меню Вид можно найти настройки внешнего вида программы, задать получение данных с сортировкой по первому столбцу, или заставить программу открываться развернутой на весь экран. Если Вы читаете этот текст - скорее всего, вы уже знаете про меню Справка. Здесь также есть окно "О программе".

Информация о выделенном элементе. Она заполняется данными из выделенной записи в заполненной библиотеке:

Кнопки действий с записями. Добавить запись, Редактировать, Удалить, Поиск записей.

Информация о выделенной библиотеке. Показывает название библиотеки и количество запией в ней. Здесь же кнопка выбора библиотек

Статусная строка. Отображает уведомления

Таблица элементов библиотеки. Достает всю информацию из библиотеки и отображает в виде строк с записями. Сортируется по столбцам
