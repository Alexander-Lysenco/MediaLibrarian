Добавление записей в библиотеку

Для добавления новой записи в библиотеку, нажмите кнопку "Добавить" на панели действий. Появится окно ввода данных для новой записи, основанное на полях, включенных в библиотеку:

Первое поле зарезервировано под название, и оно обязательно должно быть заполнено. Осталные поля поля заполняются по Вашему усмотрению информацией, соответствующей этим полям:

В полях рейтинга (оценка, приоритет) для активации поля необходимо нажать на одну из звездочек/кубиков. Рейтинг установится в нужную позицию и окрасится в соответствующий цвет. Если поле рейтинга не было тронуто, по умолчанию устанавливается нулевой.
Если рейтинг был изменен, то для установки нулевого рейтинга нажмите один раз на первую звездочку/кубик, затем еще раз для аннулирования рейтинга.

После оформления всех необходимых полей, нажмите кнопку "Сохранить" для добавления записи.

Обращаем Ваше внимание, что в библиотеке невозможно расположение нескольких записей с одинаковыми именами. Поэтому, при совпадении названия новой записи с названием какой-нибудь из уже существующих записей, Вы увидите соответствующее сообщение об ошибке.
