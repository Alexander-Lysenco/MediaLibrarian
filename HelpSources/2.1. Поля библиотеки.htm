Как строится библиотека

В программе реализовано 7 типов полей для библиотек:

  1. Строка - текстовое поле в одну строку. Ограничение строки - 128 символов. Отображается как обычная текстовая надпись. Использовать рекомендуется для пометок, тегов или значений с небольшим количеством символов
  2. Текст - текстовое поле с многострочным режимом. Ограничение поля - 32768 символов. В редакторе отображается как текстовое поле в 4 строки. В библиотекаре отображается как текстовый абзац с учетом переноса строк. В таблице отображается строкой, переносы не учитываются.
  3. Дата - поле выбора даты. По умолчанию устанавливается текущая. Дата отображается в формате число-имя_месяца-год. Пример: 1 января 1970г. Отображается как текстовая надпись.
  4. Дата + Время - поле, аналогично предыдущему, с возможностью указать время с точностью до секунд. По умолчанию устанавливаются текущие. Формат: 1.01.1970 23:59:59. Отображается как текстовая надпись
  5. Оценка(5) - поле установки рейтинга в виде звездочек. В этом поле 5 звездочек, по умолчанию устанавливается 0. В редакторе отображается как блок с кликабельными звездочками, меняющими свой цвет в зависимости от выбора рейтинга:
    • 1-2 звезды - красный
    • 3 звезды - желтый
    • 4-5 звезд - зеленый.
    Цвет меняется при клике на звездочках. Если выбрана первая звездочка, и на ней кликнуть снова, рейтинг сбрасывается (т.е. будет выбрано 0 звезд). Отображается в библиотекаре как текстовая надпись цвета выбранного рейтинга в виде пяти звездочек. Выбранные звездочки закрашены, остальные незакрашены. Если выбрано 0 звездочек, цветом не подкрашивается
  6. Оценка(10) - аналогичное предыдущему поле установки рейтинга. Отличия этого поля в том, что здесь звездочек 10. По умолчанию устанавливается 0. Цвет меняется в следующей зависимости:
    • 1-4 звезды - красный
    • 5-8 звезд - желтый
    • 9-10 звезд - зеленый.
  7. Приоритет - поле в виде полосы, зарашиваемой квадратиками. Максимальное количество квадратиков - 10, по умолчанию устанавливается 0. Если выбран первый квадратик, и на нем кликнуть снова, "приоритет" сбрасывается (т.е. будет выбрано 0 квадратиков).
