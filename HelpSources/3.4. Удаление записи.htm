Удаление записей из библиотек

Чтобы удалить ненужную запись из библиотеки, выделите необходимую запись и воспользуйтесь одним из следующимх способов:

  • Нажать кнопку "Удалить" на панели действий;
  • Горячей клавишей "Del" (Delete);
  • Из меню "Правка" выбрать "Удалить запись".

После подтверждения удаления записи из диалогового окошка выбранная запись будет удалена. Вместе с ней из папки с программой будет удален постер, если он был прикреплен к записи. Обращаем Ваше внимание, что эти действия являются необратимыми, и восстановить удаленные данные будет невозможно.
