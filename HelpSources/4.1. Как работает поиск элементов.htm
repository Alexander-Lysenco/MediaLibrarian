Как работает поиск элементов

Поиск записей по библиотеке осуществляется засчет частичного совпадения по выбранному полю библиотеки. Иначе говоря, если мы не помним точно полного названия элемента, мы можем ввести в поиск его часть, и поиск выдаст все элементы, в которых выбранное поле содержит эту часть. Поиск элементов становится доступным, если в библиотеке существует не менее пяти записей. Результаты поиска отобразятся в таблице записей главного окна программы.

Если записей по результатам поиска показано слишком много, можно включить дополнительные фильтры по другим полям. В результатах поиска покажутся только те записи, в которых есть совпадения по всем активным полям.

После закрытия формы поиска в ней сохранятся введенные Вами данные. Для их очистки воспользуйтесь кнопкой "Очистить данные". Имейте в виду, что результаты поиска также сохраняются в таблице. Для загрузки в нее всех имеющихся записей библиотеки, нажмите кнопку "Сбросить результаты".
