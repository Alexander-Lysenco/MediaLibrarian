Технологии, использованные в разработке программы

Приложение разработано на платформе Microsoft .NET Framework 4.0 (Windows Forms Application). В приложении были использованы модули System.Windows.Forms, System.Collections.Generic, System.Drawing, System.IO, System.Linq, System.Data, System.Diagnostics, System.Text.RegularExpressions.
В приложении были также использованы сторонние модули, NuGet-пакеты: EntityFramework, System.Data.Sqlite (включая зависимости).

Настройки приложения хранятся в формате XML 1.0

Данные приложения хранятся в базе данных. Для ее организации используется движок SQLite версии 3.
