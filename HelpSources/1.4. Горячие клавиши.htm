Горячие клавиши "Медиа-библиотекаря"

Комбинация Место Описание
F2 Главное окно - Файл Открыть менеджер библиотек
Ctrl+Shift+Del Главное окно - Файл Очистить текущую библиотеку (удалить все записи)
Alt+F4 Главное окно - Файл Закрыть программу
Ctrl+N Главное окно - Правка Добавить запись в библиотеку
Ctrl+E Главное окно - Правка Редактировать запись
Del Главное окно - Правка Удалить запись
Ctrl+F Главное окно - Правка Открыть поиск
F11 Главное окно - Вид На весь экран
Ctrl+P Главное окно - Вид Открыть настройки
F1 Главное окно - Справка Вызов справки
F12 Главное окно - Справка Вызов окна "О программе"
Del Менеджер библиотек Удалить выделенную библиотеку
Ctrl+Enter Редактор библиотек Сохранить
Esc Менеджер библиотек, редактор бибилотек, Поиск Закрыть окно
