Редактирование записей в библиотеке

Для начала необходимо выделить запись для редактирования из списка всех записей библиотеки. Окно редактирования можно открыть несколькими способами:

  • Двойным кликом по записи в таблице;
  • Нажав кнопку "Редактировать" на панели действий;
  • Из меню "Правка" выбрать пункт "Редактировать запись";
  • Комбинацией клавиш Ctrl+E.

Появится окно редактирования записи. Это то же окно добавления записи, но уже заполненное имеющимися данными:

Чтобы отредактировать данные, измените значения необходимых полей и нажмите "Сохранить".

Обращаем Ваше внимание, что в библиотеке невозможно расположение нескольких записей с одинаковыми именами. Поэтому, при совпадении отредактированного названия с названием какой-нибудь из уже существующих записей, Вы увидите соответствующее сообщение об ошибке.
